# 碑文金石资料检索系统 - 零基础操作手册

欢迎使用本系统！为了确保系统在任何电脑上都能顺利运行，请按照以下步骤进行操作。

---

### 1. 准备项目环境

1. **进入项目文件夹**：
   在您的电脑上找到本项目文件夹（即包含 `app`, `data`, `scripts` 等文件夹的目录）。

2. **准备 Word 文档**：
   将您的所有碑文 Word 文档（`.docx` 格式）放入以下路径：
   `项目文件夹/data/raw_word/`
   *注意：请确保文件名不包含特殊字符，建议使用数字或简单文字命名。*

---

### 2. 安装必要的软件库

在运行程序之前，需要安装 Python 运行环境及相关的工具包。

1. **打开终端 (Terminal)**：
   - **在 VS Code 中**：点击顶部菜单 `终端` -> `新建终端`，或者快捷键 `Ctrl + ~`。
   - **在 Windows 中**：在项目文件夹地址栏输入 `cmd` 并回车。

2. **运行安装命令**（复制并运行以下命令）：
   ```bash
   pip install -r requirements.txt
   ```
   *如果安装缓慢，可以使用国内镜像源：*
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

---

### 3. 初始化数据 (解析 Word)

这一步会将 Word 文档中的文字和图片提取并存入数据库。

1. **在终端中输入以下命令**：
   ```bash
   python scripts/word_parser.py
   ```
2. **观察结果**：
   - 程序会显示 `Processing [文件名]...`。
   - 看到 `Saved record: 【编号】 - [器名]` 表示该条目已成功入库。
   - 看到 `Done.` 表示处理完毕。

---

### 4. 启动检索网站

1. **在终端中输入以下命令**：
   ```bash
   uvicorn app.main:app --reload
   ```
2. **确认运行状态**：
   - 看到 `Uvicorn running on http://127.0.0.1:8000` 表示网站已成功启动。
   - **重要提示**：运行期间请勿关闭此终端窗口。

---

### 5. 打开网页进行检索

1. **访问地址**：
   打开浏览器，在地址栏输入：`http://127.0.0.1:8000` 并回车。

2. **使用检索功能**：
   - 在搜索框输入关键词（如器名、释文内容、出土地点等）。
   - 点击“检索”或按回车键。
   - 点击搜索结果卡片查看完整属性和高清图片。

---

### 6. 资料管理与导入（进阶操作）

除了初始的脚本导入，您可以通过网页版的“管理后台”进行更方便的数据更新。

#### 6.1 进入管理后台
- 在检索首页右上角点击 **“管理后台”** 按钮。
- 或者直接访问：`http://127.0.0.1:8000/admin`。

#### 6.2 批量导入新文档与冲突处理
1. **上传文档**：
   - 在“资料导入”标签页，将新的 Word 文档（支持多个）拖入上传区，或点击按钮选择文件。
   - 系统会自动开始上传并解析。

2. **查看导入报告**：
   上传完成后，页面会显示三类结果：
   - **成功 (绿色)**：新数据已顺利写入数据库。
   - **跳过/冲突 (黄色)**：发现“器名”已存在的条目，系统会自动跳过，**不会**覆盖旧数据。
   - **错误 (红色)**：文件格式错误或其他解析问题。

3. **解决冲突**：
   - 在“冲突列表”中，您可以看到所有被跳过的条目。
   - **操作**：
     - **保持现状**：如果您不想更改旧数据，什么都不用做（系统默认已跳过）。
     - **强制覆盖**：点击红色的 **“强制覆盖”** 按钮，用新上传的数据替换数据库中的旧数据。**注意：此操作不可撤销！**

#### 6.3 数据维护与意外处理
1. **搜索与删除**：
   - 切换到“数据维护”标签页。
   - 输入关键词搜索现有资料。
   - 点击 **“删除”** 按钮可永久移除错误或重复的记录。

2. **常见意外情况处理**：
   - **错别字导致的重复**：如果数据库里有“大盂鼎”，您又上传了“大孟鼎”，系统会视为两条不同的记录。
     - **解决方法**：在“数据维护”页搜索“大孟鼎”，确认是错别字后，点击删除。然后确保 Word 文档正确后重新上传，或保留正确的“大盂鼎”。
   - **上传卡住/失败**：
     - 检查 Word 文档是否过大（建议单个文档不超过 20MB）。
     - 检查是否包含特殊格式（如加密、损坏）。
     - 刷新页面重试。
   - **覆盖后后悔**：
     - 系统不支持“撤销”。建议在覆盖前先确认旧数据确实需要更新。
     - 极端情况下，如果原有 Word 文档还在，可以删除当前条目，重新上传旧版 Word 文档。

---

### 🛠️ 常见问题排查

- **报错 `ModuleNotFoundError`**：说明第二步的安装未成功，请重新运行安装命令。
- **解析 Word 失败**：请确保文档是 `.docx` 格式，且内容包含类似 `【001】` 的编号作为分割。
- **网页无法打开**：请检查第四步的终端窗口是否报错，或者是否意外关闭了终端。
- **图片无法显示**：解析完成后，请检查 `app/static/images/` 目录下是否生成了对应的图片文件。

# 碑文金石资料检索系统 - 零基础操作手册

欢迎使用本系统！为了确保系统在任何电脑上都能顺利运行，请按照以下步骤进行操作。

---

### 1. 准备项目环境

1. **进入项目文件夹**：
   在您的电脑上找到本项目文件夹（即包含 `app`, `data`, `scripts` 等文件夹的目录）。

2. **准备 Word 文档**：
   将您的所有碑文 Word 文档（`.docx` 格式）放入以下路径：
   `项目文件夹/data/raw_word/`
   *注意：请确保文件名不包含特殊字符，建议使用数字或简单文字命名。*

---

### 2. 安装必要的软件库

在运行程序之前，需要安装 Python 运行环境及相关的工具包。

1. **打开终端 (Terminal)**：
   - **在 VS Code 中**：点击顶部菜单 `终端` -> `新建终端`。
   - **在 Windows 中**：在项目文件夹地址栏输入 `cmd` 并回车。

2. **运行安装命令**（复制并运行以下命令）：
   ```bash
   pip install -r requirements.txt
   ```
   *如果安装缓慢，可以使用国内镜像源：*
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

---

### 3. 初始化数据 (解析 Word)

这一步会将 Word 文档中的文字和图片提取并存入数据库。

1. **在终端中输入以下命令**：
   ```bash
   python scripts/word_parser.py
   ```
2. **观察结果**：
   - 程序会显示 `Processing [文件名]...`。
   - 看到 `Saved record: 【编号】 - [器名]` 表示该条目已成功入库。
   - 看到 `Done.` 表示处理完毕。

---

### 4. 启动检索网站

1. **在终端中输入以下命令**：
   ```bash
   uvicorn app.main:app --reload
   ```
2. **确认运行状态**：
   - 看到 `Uvicorn running on http://127.0.0.1:8000` 表示网站已成功启动。
   - **重要提示**：运行期间请勿关闭此终端窗口。

---

### 5. 打开网页进行检索

1. **访问地址**：
   打开浏览器，在地址栏输入：`http://127.0.0.1:8000` 并回车。

2. **使用检索功能**：
   - 在搜索框输入关键词（如器名、释文内容、出土地点等）。
   - 点击“检索”或按回车键。
   - 点击搜索结果卡片查看完整属性和高清图片。

---

### 🛠️ 常见问题排查

- **报错 `ModuleNotFoundError`**：说明第二步的安装未成功，请重新运行安装命令。
- **解析 Word 失败**：请确保文档是 `.docx` 格式，且内容包含类似 `【001】` 的编号作为分割。
- **网页无法打开**：请检查第四步的终端窗口是否报错，或者是否意外关闭了终端。
- **图片无法显示**：解析完成后，请检查 `app/static/images/` 目录下是否生成了对应的图片文件。
